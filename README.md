# 📌 LinkNLink

> 링크를 연결하고 공유하는 프로젝트

## ✨ 프로젝트 소개

이 프로젝트는 링크를 효율적으로 관리하고 공유하기 위한 목적으로 시작되었습니다.

## 🚀 주요 기능

- 🔗 링크 저장 및 관리
- 📂 카테고리별 분류
- 🔍 링크 검색
- 👥 링크 공유

## 📋 시작하기

### 필요 환경
- 웹 브라우저
- HTMLL/CSS/JavaScript 지식

### 설치 방법

```bash
git clone https://github.com/SunMookLee/LinkNLink.git
cd LinkNLink
```

## 📚 사용 방법

1. 프로젝트 폴더를 엽니다
2. `index.html` 파일을 브라우저에서 실행합니다
3. 링크를 추가하고 관리합니다
